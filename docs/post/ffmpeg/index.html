ffmpeg

— Li Xunsong Fri Apr 9, 2021
  • 使用 ffmpeg 将一系列帧图片转换成 .gif 和 .avi
ffmpeg -f image2 -i frame%4d.jpg video.avi
ffmpeg -i video.avi -t 5 out.gif

  • 将视频转换成图片

    ffmpeg -i ./video.webm ./video/image%d.jpg
