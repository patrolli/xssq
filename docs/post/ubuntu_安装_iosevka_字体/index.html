ubuntu 安装 iosevka 字体

— Li Xunsong
2021-04-09

Contents

  • 从 github 的代码仓库中下载 zip 文件，放到一个临时目录中 ("~/temp/")

  • 解压文件

    1

    unzip -u ttf-iosevka-5.2.1.zip -d iosevka

  • 放到系统的字体目录：

    1

    sudo cp -r ~/temp/iosevka /usr/share/fonts/

  • 重新刷新字体缓存

    1

    fc-cache -fv
